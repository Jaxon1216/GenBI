## 🗄️ 数据库改动

### 改动 1：chart 表添加 name 字段

**执行时间**: 2026-03-13 23:47

**修改内容**:
```sql
ALTER TABLE chart ADD COLUMN `name` varchar(128) NULL COMMENT '图表名称' AFTER id;
```

**修改原因**:
- 后端 `Chart.java` 实体类定义了 `name` 字段
- 建表脚本 `create_table.sql` 中包含 `name` 字段定义
- 但实际数据库中缺少该字段，导致查询报错：`Unknown column 'name' in 'field list'`

**验证方法**:
```sql
-- 查看表结构，确认 name 字段存在
DESCRIBE chart;

-- 预期输出应包含：
-- name | varchar(128) | YES | | NULL |
```

**当前表结构**:
```
Field       | Type         | Null | Key | Default           | Extra
------------|--------------|------|-----|-------------------|------------------
id          | bigint       | NO   | PRI | NULL              | auto_increment
name        | varchar(128) | YES  |     | NULL              |
goal        | text         | YES  |     | NULL              |
chartData   | text         | YES  |     | NULL              |
chartType   | varchar(128) | YES  |     | NULL              |
genChart    | text         | YES  |     | NULL              |
genResult   | text         | YES  |     | NULL              |
userId      | bigint       | YES  |     | NULL              |
createTime  | datetime     | NO   |     | CURRENT_TIMESTAMP | DEFAULT_GENERATED
updateTime  | datetime     | NO   |     | CURRENT_TIMESTAMP | DEFAULT_GENERATED on update
isDelete    | tinyint      | NO   |     | 0                 |
```

## 💻 后端代码改动

### 改动 1：`AiManager.java` - AI 接口改造（整个文件替换）

**文件路径**: `backend/src/main/java/com/yupi/springbootinit/manager/AiManager.java`

**改动原因**: 鱼聪明平台已停止服务，改用百度千帆 OpenAI 兼容接口

**改动内容**: 
- 移除鱼聪明 SDK 依赖
- 使用 hutool HttpRequest 直接调用 HTTP API
- 方法签名从 `doChat(modelId, message)` 改为 `doChat(systemPrompt, userMessage)`

**详细改动**: 参见 `AI_MIGRATION_GUIDE.md` 第 78-245 行

---

### 改动 2：`ChartController.java` - 图表生成接口调整

**文件路径**: `backend/src/main/java/com/yupi/springbootinit/controller/ChartController.java`

**改动原因**: 配合 AiManager 接口改造

**关键改动**:
```java
// ❌ 旧代码（使用模型 ID）
long biModelId = 1659171950288818178L;
String result = aiManager.doChat(biModelId, userInput.toString());
String[] splits = result.split("【【【【【");

// ✅ 新代码（使用 system prompt）
final String systemPrompt = "你是一个数据分析师和前端开发专家...";
String result = aiManager.doChat(systemPrompt, userInput.toString());
String[] splits = result.split("【【【【");
```

**改进点**:
- 优化了 AI prompt，提高生成质量
- 改变分隔符从 5 个【 改为 4 个【
- 直接在代码中定义 prompt，不依赖外部模型 ID

### 问题 2: name 字段仍然报错

**错误信息**:
```
Unknown column 'name' in 'field list'
```

**解决方案**:
```sql
-- 1. 确认字段是否存在
DESCRIBE chart;

ALTER TABLE chart ADD COLUMN `name` varchar(128) NULL COMMENT '图表名称' AFTER id;

```

### 问题 3: Maven 依赖下载失败

**错误信息**:
```
Could not resolve dependencies
```

**解决方案**:
```bash
# 1. 清理本地仓库
mvn clean

# 2. 强制更新依赖
mvn clean install -U

# 3. 如果还不行，删除本地仓库缓存
rm -rf ~/.m2/repository
mvn clean install
```

### 问题 4: 端口被占用

**错误信息**:
```
Port 12345 is already in use
```

**解决方案**:
```bash
# 1. 查找占用端口的进程
lsof -i :12345

# 2. 杀死进程
kill -9 <PID>

# 3. 或者修改端口号
# 编辑 application.yml
server:
  port: 12346  # 改为其他端口
```
